How To Install:

View on GitHub Download

Install from Bower

$ bower install patch-panel

How to Use:

HTML

First, include the path to the file, remember to include jQuery beforehand.

<script src=/path/to/jQuery.min.js"><script>
<script src=/path/to/patchpanel.min.js"><script>

Patch Panel works on the parent container of a group of items and panels.

<div class="patch-container">
    <div class="patch-item">...<div>
    <div class="patch-panel">...<div>
    <div class="patch-item">...<div>
    <div class="patch-panel">...<div>
    ...
</div>

There are four different components that make up Patch Panel's magic.

.patch-item Place this class on all child items that you want to be shown.

.patch-panel Place this class on all child items that are panels.

.patch-button Place this class on that element you want to use to trigger.

data-patch-panel Place this data attribute along with a unique identifier on both your patch-button and your patch-panel, this is what ties the two together.

CSS

All sizing of items is handled by your CSS

.patch-item {
  height: 200px;
  width: 100%;
}

@media only screen and (min-width : 992px) {
  .patch-item {
    width: 50%;
  }
}

jQuery Initialization

$('.patch-container').patchpanel({
  buttonSelector: '.grid-button',
  itemSelector: '.grid-item',
  panelSelector: '.grid-panel'
});
